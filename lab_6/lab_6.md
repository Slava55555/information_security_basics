### 2. Исходные данные для варианта №3

|----------|----------|

### 3. Расчёт эффективности экранирования

#### 3.1. Определение типа экранирования и зоны наблюдения
Длина волны излучения:
`λ = V / f = 3*10^8 / (300*10^3) = 1000 м`

Граница ближней/дальней зоны:
`5λ = 5 * 1000 = 5000 м`

Заданное расстояние (эквивалентный радиус):
`R_э = 0,7 м << 5000 м`

**Вывод:** Работа ведётся в **ближней зоне**. Поскольку материал немагнитный и частота относительно высока, расчёт проводится для **электростатического экранирования**.

#### 3.2. Расчёт эффективности электростатического экранирования
Используется формула для электростатического экранирования плоского листового экрана замкнутой формы (таблица 1):

`S_E = 20 * lg(1 + d/δ) [дБ]`

где `δ` — эквивалентная глубина проникновения (скин-слой):

`δ = √(ρ / (π * f * μ_0 * μ_r))`

**Расчёт δ:**
- `ρ = 4,7×10⁻⁸ Ом·м`
- `f = 300×10³ Гц`
- `μ_0 = 4π×10⁻⁷ Гн/м`
- `μ_r = 1`

`δ = √(4,7*10^-8 / (π * 300*10^3 * 4π*10^-7 * 1))`

`δ ≈ √(4,7*10^-8 / 1,183) ≈ √(3,97*10^-8) ≈ 1,99*10^-4 м`

**Расчёт S_E:**
`S_E = 20 * lg(1 + 0,002 / (1,99*10^-4)) = 20 * lg(1 + 10,05) ≈ 20 * lg(11,05)`
`lg(11,05) ≈ 1,043`
`S_E ≈ 20 * 1,043 ≈ 20,86 дБ`

**Результат:**  
Расчётная эффективность электростатического экранирования составляет **≈20,86 дБ**, что **ниже** заданного значения 28 дБ.

### 4. Определение требуемой толщины экрана для достижения заданной эффективности (28 дБ)

Из формулы:
`28 = 20 * lg(1 + d/δ)`
`1,4 = lg(1 + d/δ)`
`10^1,4 ≈ 25,12 = 1 + d/δ`
`d/δ ≈ 24,12`
`d ≈ 24,12 * δ ≈ 24,12 * 1,99*10^-4 ≈ 0,0048 м = 4,8 мм`

**Вывод:** Для достижения эффективности 28 дБ при частоте 300 кГц и материале стеклотекстолит требуется толщина стенки экрана **≈4,8 мм**.

---

### 5. Расчёт геометрических размеров экранов различной формы

При заданном эквивалентном радиусе `R_э = 0,7 м`.

#### 5.1. Прямоугольный экран
Формула:
`R_э = ∛(l₁ * l₂ * l₃ / (4π))`

Примем кубическую форму (`l₁ = l₂ = l₃ = a`):
`0,7 = ∛(a³ / (4π))`
`a³ = 0,7³ * 4π ≈ 0,343 * 12,566 ≈ 4,31`
`a ≈ ∛4,31 ≈ 1,63 м`

#### 5.2. Цилиндрический экран
Формула:
`R_э = ∛(D² * h / 4)`

Примем `D = h`:
`0,7 = ∛(D³ / 4)`
`D³ = 0,7³ * 4 ≈ 0,343 * 4 ≈ 1,372`
`D ≈ ∛1,372 ≈ 1,11 м`

#### 5.3. Сферический экран
`R_э = r_сф = 0,7 м`

**Результаты:**

| Форма экрана | Размеры |
|----------|----------|
| Прямоугольный (куб) | `a ≈ 1,63 м` |
| Цилиндрический (D = h) | `D ≈ 1,11 м`, `h ≈ 1,11 м` |
| Сферический | `r ≈ 0,7 м` |

### 6. Выводы

1. Для материала стеклотекстолит на частоте 300 кГц эффективность электростатического экранирования при толщине стенки 2 мм составляет **≈20,86 дБ**, что не удовлетворяет заданному требованию (28 дБ).
2. Для достижения требуемой эффективности необходимо увеличить толщину стенки экрана до **≈4,8 мм**.
3. При заданном эквивалентном радиусе 0,7 м определены следующие размеры экранов:
    - Прямоугольный кубический экран: сторона ≈1,63 м
    - Цилиндрический экран: диаметр и высота ≈1,11 м
    - Сферический экран: радиус 0,7 м
4. Эффективность экранирования существенно зависит от толщины материала, его удельного сопротивления и геометрической формы экрана. Для повышения эффективности целесообразно использовать материалы с меньшим удельным сопротивлением (медь, алюминий) или увеличивать толщину стенок.
