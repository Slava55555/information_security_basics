## 3. Расчет нормализованных весов

### 3.1. Геометрическое среднее по строкам

Формула: `Gᵢ = ⁿ√∏aᵢⱼ`, где `n = 6`

- `G₁ = ⁶√(1×4×3×2×6×5) ≈ 2.99`
- `G₂ = ⁶√(1/4×1×1/2×1/3×4×3) ≈ 0.86`
- `G₃ = ⁶√(1/3×2×1×1/2×5×4) ≈ 1.29`
- `G₄ = ⁶√(1/2×3×2×1×6×5) ≈ 2.05`
- `G₅ = ⁶√(1/6×1/4×1/5×1/6×1×1/2) ≈ 0.35`
- `G₆ = ⁶√(1/5×1/3×1/4×1/5×2×1) ≈ 0.49`

Сумма `∑Gᵢ ≈ 8.03`

### 3.2. Нормализация

`wᵢ = Gᵢ / ∑Gᵢ`

- `w₁ ≈ 2.99 / 8.03 ≈ 0.37`
- `w₂ ≈ 0.86 / 8.03 ≈ 0.11`
- `w₃ ≈ 1.29 / 8.03 ≈ 0.16`
- `w₄ ≈ 2.05 / 8.03 ≈ 0.26`
- `w₅ ≈ 0.35 / 8.03 ≈ 0.04`
- `w₆ ≈ 0.49 / 8.03 ≈ 0.06`

## 4. Проверка согласованности оценок

### 4.1. Вычисление λ_max

1. Суммы по столбцам матрицы:
    - `S₁ = 1 + 1/4 + 1/3 + 1/2 + 1/6 + 1/5 ≈ 2.45`
    - `S₂ = 4 + 1 + 2 + 3 + 1/4 + 1/3 ≈ 10.58`
    - `S₃ = 3 + 1/2 + 1 + 2 + 1/5 + 1/4 ≈ 6.95`
    - `S₄ = 2 + 1/3 + 1/2 + 1 + 1/6 + 1/5 ≈ 4.20`
    - `S₅ = 6 + 4 + 5 + 6 + 1 + 2 ≈ 24.00`
    - `S₆ = 5 + 3 + 4 + 5 + 1/2 + 1 ≈ 18.50`

2. `λ_max = ∑(Sⱼ × wⱼ)`
    - `λ_max ≈ 2.45×0.37 + 10.58×0.11 + 6.95×0.16 + 4.20×0.26 + 24.00×0.04 + 18.50×0.06 ≈ 6.35`

### 4.2. Индекс согласованности (ИС)

`ИС = (λ_max - n) / (n - 1) = (6.35 - 6) / 5 = 0.07`

### 4.3. Отношение согласованности (ОС)

Для `n = 6` случайная согласованность `СИ = 1.24`

`ОС = ИС / СИ = 0.07 / 1.24 ≈ 0.056 ≈ 5.6%`

## 5. Выводы

1. Наиболее важным требованием для снижения уровня риска является **Требование 3** (нормализованный вес ≈ 0.37), что связано с комплексной защитой охраняемого периметра.
2. Вторым по значимости идёт **Требование 29** (≈ 0.26), связанное с защитой кабельных систем.
3. Наименьший вес имеет **Требование 32** (≈ 0.04), что может объясняться его узкой специализацией на промышленных зонах.
4. Отношение согласованности (ОС = 5.6%) находится в допустимых пределах (менее 10%), что свидетельствует о **высокой согласованности экспертных оценок**.
5. Метод парных сравнений Т.Л. Саати позволяет объективно ранжировать требования безопасности на основе экспертных суждений.
